MetaBBS 설치하기

시작하기 전에

MetaBBS는 PHP와 MySQL을 지원하는 서버에만 설치할 수 있습니다. 가급적이면 PHP 5와 MySQL 4.1 이상을 권장합니다.

1. 파일 올리기

설치를 진행하기 위해서는 먼저 MetaBBS를 서버에 업로드해야 합니다. SSH나 텔넷을 이용할수도 있지만, 이 도움말에서는 FTP를 사용하여 올리는 방법에 대해서만 설명합니다.

  1. 먼저 MetaBBS의 배포판을 내려받습니다. MetaBBS 홈페이지에서 받을 수 있습니다.
  2. 받은 파일의 압축을 풉니다. 평소에 자주 사용하시던 프로그램을 쓰시면 됩니다.
  3. FTP 클라이언트를 켜고 MetaBBS를 설치할 서버에 접속하여, 압축을 풀어 나온 파일들을 설치할 디렉토리에 업로드합니다.

2. 퍼미션 조정하기

파일을 모두 올렸다면, 인스톨러에 접속해봅니다. 웹 브라우저를 켜고 http://MetaBBS가 설치된 URL/install.php에 접속하면 됩니다. 그러면 이런 메시지가 나올겁니다.

MetaBBS를 설치하고 사용하기 위해서는 웹 서버가 해당 디렉토리에 쓰기 권한을 가지고 있어야 하는데, 그럴 수 있는 권한이 없기 때문에 나오는 메시지입니다. FTP 클라이언트로 MetaBBS 디렉토리의 퍼미션을 707로 변경합니다. (아래 화면은 FileZilla에서의 퍼미션 변경 화면입니다.)

참고: 일부 서버에서는 이 과정이 필요 없을수도 있습니다. 그렇다면 곧바로 다음 과정으로 넘어가십시오.

3. 설치하기

위의 과정을 마쳤다면 인스톨러를 새로 고침 해봅니다. 다음과 같은 화면이 나오면 성공한 것입니다.

이제 데이터베이스 정보를 입력합니다. 잘 모르겠다면, 서버 관리자에게 문의해 보십시오.

마지막으로 관리자 정보를 입력한 다음 [설치] 버튼을 누릅니다.

다음과 같은 화면이 나오면 설치가 성공적으로 완료된 것입니다. 이제 Go to administration page »를 클릭하여 관리자 화면으로 이동하면 됩니다.
